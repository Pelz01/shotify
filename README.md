# Shotify

A screenshot beautifier web app that transforms plain screenshots into polished, professional images with customizable backgrounds, shadows, and styling.

## Features

- **Drag and drop** - Upload images by dragging and dropping or clicking to browse
- **Gradients** - 20 premium gradient backgrounds including Nebula, Sunset, Aurora, and more
- **Solid colors** - 20 solid color options from pure white to vibrant hues
- **Adjustable padding** - Control the spacing around your image (16-150px)
- **Shadow control** - Fine-tune the drop shadow intensity (0-100%)
- **Border radius** - Customize corner roundness (0-48px)
- **High-quality export** - Download images as PNG at 3x resolution

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- html-to-image
- Lucide React

## Getting Started

### Prerequisites

- Node.js 18+
- npm, yarn, pnpm, or bun

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
npm start
```

## Usage

1. Drop an image onto the upload area or click to select a file
2. Choose a gradient or solid color background
3. Adjust padding, shadow, and corner radius using the sliders
4. Click Download to export your beautified screenshot
